# Assignments

Each lesson contains its own assignment (see the lesson file). This directory
holds the materials that span lessons:

| File | Purpose |
|------|---------|
| `capstone-brief.md` | Full Lesson 15 capstone specification |
| `submission-template.md` | Per-lesson deliverable submission format |
| `capstone-submission-template.md` | The capstone submission map |

All deliverables are committed to your Git repo and reference the ticket ID
from the lesson.
