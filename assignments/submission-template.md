# Lesson Deliverable — Submission Template

Copy this for each lesson assignment. Keep it in your repo alongside the work.

```markdown
# <TICKET-ID> — <Lesson title> Deliverable

## Summary
One paragraph: what the ticket asked and what I delivered.

## Evidence
- Commands run (and why).
- Key output (paste or screenshot, redact secrets).
- Before/after state where relevant.

## Predict → run → explain
- Prediction before key command(s): …
- What actually happened: …
- Plain-language explanation: …
- How I verified it: …

## Decisions & tradeoffs
What I chose and why; alternatives I rejected.

## AI workflow
Asked: … | Right: … | Wrong/corrected: … | Verified with: …

## Definition of Done
- [ ] criterion 1
- [ ] criterion 2
- [ ] criterion 3

## Reflection
Answers to the lesson's reflection questions.
```

**Grading note:** see [`../ASSESSMENT_RUBRIC.md`](../ASSESSMENT_RUBRIC.md) for
weights. Documentation and reproducibility count as much as correctness.
